## Comandos interessantes para iniciantes no Git

### Atualizar o projeto local

Baixa as alterações mais recentes do repositório remoto para a sua máquina.

```bash
git pull
```

### Criar uma nova branch

Cria uma nova branch e já muda para ela. Troque `nome-da-sua-branch` pelo nome desejado.

```bash
git checkout -b nome-da-sua-branch
```

### Adicionar arquivos modificados

Adiciona todos os arquivos alterados para serem incluídos no próximo commit.

```bash
git add .
```

### Criar um commit

Salva as alterações adicionadas com uma mensagem descritiva.

```bash
git commit -m "feat: adiciona nova funcionalidade"
```

### Enviar alterações para o GitHub

Envia as alterações da sua branch local para o repositório remoto.

```bash
git push -u origin nome-da-sua-branch
```

### Exemplo para enviar na branch principal

Use apenas se você estiver trabalhando diretamente na branch `main`.

```bash
git push -u origin main
```
